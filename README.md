# Library Management System

## Technologies Used
- Angular 17 + Html + Css(Frontend)
- ###Backend###
- Node.js + Express
- MongoDB
- JWT Authentication

## Features
- User authentication (Login/Register)
- Role-based access control (Admin / Member)
- Book borrowing & returning
- Protected routes
- Access matrix for admins

## How to Run the Project

### Backend
```bash
cd lbs-backend
npm install
npm run dev
##Frontend###
